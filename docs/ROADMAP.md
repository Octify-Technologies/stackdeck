# Roadmap

## Now

- Bespoke premium templates (case-study, pitch, editorial), not recolors of generic components
- Real `::grid` and `::cell` primitives with span control and asymmetric splits
- Image support with bleed, captions, and aspect-ratio control
- Per-template typography pairings and a non-linear type scale
- Polish primitives: slide numbers, kickers, footers, section markers
- More directives for premium element types: `::testimonial`, `::logo-wall`, `::big-number`, `::comparison`, `::timeline`, `::process`, `::code` (syntax highlighted), `::pull-quote` with avatar, plus charts, tables, callouts, badges, dividers
- Brand kit per project: logo, colors, fonts, footer applied across every deck in the project
- Drag-to-reorder slides in the thumbnail panel
- Undo and redo
- Slide library: save any slide as a reusable block, drop into other decks
- Template gallery with rendered preview cards and palette swap at pick time
- Projects and folders to organize decks (e.g. "Octify Case Studies")
- In-preview editing: click any element in the slide preview to edit in place

## Next

- Custom font upload (woff2)
- Speaker notes via `::notes` directive
- Fullscreen present mode with arrow-key nav and timer
- Version history via auto-snapshots in IndexedDB
- Duplicate deck and duplicate slide
- User-authored templates with `.template.json` import and export
- CodeMirror editor with `/` directive palette and live syntax highlighting
- Find and replace across deck
- Outline view
- Keyboard shortcuts (cmd+/, cmd+d, cmd+1..9)
- Color contrast linter against the active palette
- Required alt text on images

## Later

- AI assist inside the editor: rewrite slide, suggest layout, draft section
- Sharing and growth: public deck pages, embed badge, showcase gallery
- Public stable grammar and template marketplace (v1.0)
